# Navegação Autônoma — Pequi Mecânico @Work 2026

Simulação de navegação autônoma desenvolvida para o processo seletivo do Núcleo de Robótica Pequi Mecânico.

## 🎥 Demonstração
https://youtu.be/EQxHaZl51es

## 🛠️ Tecnologias utilizadas
- ROS 2 Kilted
- Nav2 — navegação autônoma
- SLAM Toolbox — mapeamento
- Gazebo Ionic — simulação
- RViz — visualização
- Turtlebot3

## 📋 Descrição
Sistema de patrulha e vistoria autônoma para ambiente industrial, capaz de navegar desviando de obstáculos estáticos e dinâmicos usando costmaps, AMCL e planejamento de rotas com Nav2.
